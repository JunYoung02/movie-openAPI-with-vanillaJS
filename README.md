# CINE

실시간 영화 정보 제공 서비스

## What is CINE ?

카테고리 별 순위, 평점 정보를 실시간으로 업데이트하여 사용자에게 전달합니다.

- 카테고리
    - 평점 순으로 정렬된 영화
    - 개봉 예정 영화 영화
    - 실시간 상영 영화
- 각 영화별 상세 정보
    - 장르
    - 개봉일자
    - 등장인물
    - 줄거리
    - 예고편
- 영화 검색
